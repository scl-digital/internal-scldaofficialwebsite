// Installation guide

1.Open scl_email_signature.html in Chrome or Safari
2.Visually drag-select the signature card (banner → bottom bar) and Copy
3.Gmail: Settings → See all settings → Signature → New → Paste
4.Outlook: Settings → View all → Compose & reply → Signature → Paste
5.Replace placeholder values: email · phone · website · all social URLs
6.Profile photo: replace avatar div with <img src="photo.jpg" width="72" height="72" style="border-radius:50%;">
7.Logo: replace logo-placeholder div with <img src="logo.png" width="52" height="52" style="border-radius:8px;">
