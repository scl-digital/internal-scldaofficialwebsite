MANAGING PLANS (Admin Side)
─────────────────────────────────────────────────────────────────
URL: http://localhost/softwarecreativelabs.com-updates/AdminPortal/admin.html

STEP 1: Navigate to Products & Services
  → Click sidebar menu
  → Select "Products & Services"
  → Page loads with all existing services

STEP 2: Create a New Service
  → Click "+ Add Service" button
  → Fill in the form:
  
  REQUIRED FIELDS:
  ├─ Service Name: (e.g., "DIGIBOOST PRO")
  └─ Price (ZAR): (e.g., 11800)
  
  OPTIONAL FIELDS:
  ├─ Billing Period: Select month/year/one-time
  ├─ Description: Brief description (max 120 chars shown)
  ├─ Image URL: Path to plan logo/image
  └─ Details URL: Link to full details page
  
  FEATURES (What you get):
  ├─ Enter each feature on a NEW LINE
  ├─ Example:
  │   Unlimited projects
  │   24/7 Support
  │   Custom integrations
  │   Dedicated manager
  └─ Each becomes a checkmark (✓) in the display
  
  NOTES (What else you should know):
  ├─ Enter each note on a NEW LINE
  ├─ Example:
  │   Annual billing = 10% discount
  │   Cancel with 30-day notice
  │   Enterprise SLA included
  └─ Each becomes a bullet point (•) in the display
  
  STATUS:
  └─ Check "Active" to make plan visible to customers

STEP 3: Save Service
  → Click "Save Service" button
  → See success message
  → Modal closes automatically
  → Service appears in grid

STEP 4: Edit Existing Service
  → Click on any service card in the grid
  → Modal opens with current data
  → Make changes needed
  → Click "Save Service"

STEP 5: Delete Service
  → Click on service card to open it
  → Click "Delete" button (appears only when editing)
  → Confirm deletion
  → Service marked as inactive

FORM FIELD REFERENCE
─────────────────────────────────────────────────────────────────

Service Name *
├─ What: Plan/service name (displayed on card)
├─ Example: "DIGIBOOST ENTERPRISE"
└─ Required: YES

Price (ZAR) *
├─ What: Price without currency symbol
├─ Example: 29500
└─ Required: YES

Billing Period
├─ Options: month / year / one-time
├─ Default: month
├─ Shows as: "ZAR 29,500 /month"
└─ Optional

Description
├─ What: Brief plan description (first 120 chars shown on card)
├─ Example: "Complete digital solutions for enterprises"
└─ Optional

Image URL
├─ What: Path to plan logo/image
├─ Example: "/images/enterprise.png"
└─ Optional

Details URL
├─ What: Link when "See full details" is clicked
├─ Example: "/services-platform/enterprise-details"
└─ Optional

Features (one per line)
├─ What: List of plan features/benefits
├─ How: Type each feature on a new line
├─ Display: Shows with checkmark (✓) on card
├─ Example:
│   Unlimited projects
│   24/7 Priority support
│   Custom integrations
│   Dedicated account manager
└─ Optional

Notes (one per line)
├─ What: Important information/restrictions
├─ How: Type each note on a new line
├─ Display: Shows with bullet point (•) on card
├─ Example:
│   Billed annually for 10% discount
│   Cancel anytime with 30-day notice
│   Enterprise SLA included
└─ Optional

Active (Checkbox)
├─ What: Controls if plan is visible to customers
├─ Checked = Visible in "Choose Your Plan" section
├─ Unchecked = Hidden from customers
├─ Default: Checked
└─ Optional

TROUBLESHOOTING
─────────────────────────────────────────────────────────────────

❌ Form won't submit
  → Ensure Service Name and Price are filled
  → Check that price is a valid number
  → Try clearing form and re-entering data

❌ Changes not visible on site
  → Clear browser cache (Ctrl+Shift+Delete)
  → Hard refresh page (Ctrl+F5)
  → Check that "Active" checkbox is marked

❌ Features/Notes not showing
  → Make sure each item is on a separate line
  → No empty lines between items
  → Check browser console for errors (F12)

❌ Can't access admin panel
  → Verify you're logged in
  → Check URL is correct
  → Clear cookies and login again

API ENDPOINTS (For Developers)
─────────────────────────────────────────────────────────────────

GET /api/services.php
└─ Returns: All active services with all fields

GET /api/services.php?id=1
└─ Returns: Single service by ID

GET /api/admin/services.php
└─ Returns: All services (admin view)

POST /api/admin/services.php
├─ Creates new service
└─ Body: { name, price_amount, ... }

PUT /api/admin/services.php
├─ Updates existing service
└─ Body: { id, name, price_amount, ... }

DELETE /api/admin/services.php
├─ Deactivates service
└─ Body: { id }
